## Instalacion

Instale las dependencias y devDependencies e inicie el servidor.

```sh
cd IndicadoresApp
npm i / yarn install
yarn android
```

For ios...

```sh
cd IndicadoresApp
cd ios
pod install
return to IndicadoresApp
yarn ios
```
